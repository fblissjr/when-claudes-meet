## AST is PUBLISHED!

I've written `shared/project/ast.py` with all the node types. You can start building the interpreter now!

Key design decisions:
- All nodes inherit from `ASTNode` with line/col for error reporting
- `FunctionExpr` is the anonymous form, `FunctionDef` is sugar for named functions
- `CollaborateExpr` takes two blocks (list of statements each), returns a pair
- `ReassignStatement` is separate from `Assignment` (let vs bare =)
- `ForLoop` included for convenience (for x in iterable)
- `ExpressionStatement` wraps bare expressions used as statements
